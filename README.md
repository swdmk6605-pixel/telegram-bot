# 🤖 بوت إدارة اشتراكات القناة

## المميزات
- ✅ إضافة مشتركين مع رابط دعوة خاص لكل واحد
- ✅ خطط: يومي / أسبوعي / شهري / سنوي
- ✅ طرد تلقائي عند انتهاء الاشتراك
- ✅ تنبيه قبل 3 أيام وقبل يوم من الانتهاء
- ✅ لوحة تحكم للأدمن
- ✅ إحصائيات وبحث

## خطوات الرفع على Railway

### 1. احصل على ID حسابك
- ابحث عن @userinfobot بتلغرام
- أرسل له /start
- راح يعطيك رقمك (مثل: 123456789)

### 2. ارفع على Railway
1. اذهب إلى railway.app
2. سجل دخول بـ GitHub
3. اضغط "New Project"
4. اختر "Deploy from GitHub repo"
5. ارفع الملفات أو استخدم GitHub

### 3. أضف المتغيرات
في Railway اضغط Variables وأضف:
- BOT_TOKEN = توكنك الجديد
- CHANNEL_ID = -1003990153810
- ADMIN_ID = رقم حسابك

### 4. شغّل البوت
Railway راح يشغله تلقائياً!

## كيف تستخدمه
1. افتح البوت بتلغرام
2. اكتب /start
3. تظهر لوحة التحكم
4. اضغط "إضافة مشترك جديد"
5. أدخل ID المشترك واختر الخطة
6. البوت يولد رابط خاص - أرسله للمشترك
